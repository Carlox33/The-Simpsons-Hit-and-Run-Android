## &#x1F1EA;&#x1F1F8; Espa&#x00F1;ol — Controles para mando externo  &#x1F3AE;

Se puede jugar con diversos mandos, pero aquí se utilizarán los nombres de los botones de PlayStation como ejemplo para ilustrar los controles.  
Se ha probado tanto con **DualShock 4 (mando de PS4)** como con **GameSir X2 Pro**, y ambos comparten las mismas asignaciones, por lo que salvo contadas excepciones estos son los controles del juego.

### General
- **Start**: Pausa
- **Seleccionar (OK)**: X
- **Hacia atrás**: Círculo

### A pie
- **Movimiento del personaje**: Cruceta o joystick izquierdo
- **Correr**: Círculo
- **Saltar**: X
- **Doble salto**: X + X
- **Golpear**: Cuadrado
- **Interactuar**: Triángulo
- **Mover cámara**: Joystick derecho
- **Zoom in**: Joystick derecho hacia adelante o L2 (más intenso)
- **Zoom out**: Joystick derecho hacia atrás
- **Mirar hacia arriba**: R2

### En vehículo
- **Acelerar**: R2 (progresivo) o X
- **Frenar**: L2 (progresivo) o Círculo
- **Cambiar cámara**: L1
- **Claxon**: R1
- **Mover cámara**: Joystick derecho
- **Mirar hacia atrás**: Joystick derecho hacia atrás
- **Resetear posición**: Select

# Game Controls &#x1F3AE;

## &#x1F1EC;&#x1F1E7; English — External Controller Controls &#x1F3AE;

The game can be played with a variety of controllers, but the **PlayStation button names** are used here as examples to illustrate the controls.  
The game has been tested with both a **DualShock 4 (PS4 controller)** and the **GameSir X2 Pro**, and both share the same mappings, so except for a few cases these are the game controls.

### General
- **Start**: Pause
- **Select (OK)**: X
- **Back**: Circle

### On Foot
- **Character movement**: D-pad or left joystick
- **Run**: Circle
- **Jump**: X
- **Double jump**: X + X
- **Attack**: Square
- **Interact**: Triangle
- **Move camera**: Right joystick
- **Zoom in**: Push right joystick forward or L2 (stronger)
- **Zoom out**: Pull right joystick backward
- **Look up**: R2

### In Vehicle
- **Accelerate**: R2 (progressive) or X
- **Brake**: L2 (progressive) or Circle
- **Change camera**: L1
- **Horn**: R1
- **Move camera**: Right joystick
- **Look back**: Pull right joystick backward
- **Reset position**: Select

# Commandes du jeu &#x1F3AE;

## &#x1F1EB;&#x1F1F7; Fran&#x00E7;ais — Commandes pour manette externe &#x1F3AE;

Le jeu peut être joué avec différents contrôleurs, mais les **noms des boutons PlayStation** sont utilisés ici comme exemple pour illustrer les commandes.  
Le jeu a été testé avec une **DualShock 4 (manette PS4)** et une **GameSir X2 Pro**, qui partagent les mêmes assignations. À quelques exceptions près, voici les commandes du jeu.

### Général
- **Start** : Pause
- **Select (OK)** : X
- **Retour** : Cercle

### À pied
- **Déplacement du personnage** : Croix directionnelle ou joystick gauche
- **Courir** : Cercle
- **Sauter** : X
- **Double saut** : X + X
- **Attaquer** : Carré
- **Interagir** : Triangle
- **Déplacer la caméra** : Joystick droit
- **Zoom avant** : Joystick droit vers l’avant ou L2 (plus intense)
- **Zoom arrière** : Joystick droit vers l’arrière
- **Regarder vers le haut** : R2

### En véhicule
- **Accélérer** : R2 (progressif) ou X
- **Freiner** : L2 (progressif) ou Cercle
- **Changer la caméra** : L1
- **Klaxon** : R1
- **Déplacer la caméra** : Joystick droit
- **Regarder en arrière** : Joystick droit vers l’arrière
- **Réinitialiser la position** : Select

# Spielsteuerung &#x1F3AE;

## &#x1F1E9;&#x1F1EA; Deutsch — Steuerung für externen Controller &#x1F3AE;

Das Spiel kann mit verschiedenen Controllern gespielt werden, aber hier werden die **PlayStation-Tastenbezeichnungen** als Beispiel verwendet, um die Steuerung zu erklären.  
Das Spiel wurde sowohl mit einem **DualShock 4 (PS4-Controller)** als auch mit dem **GameSir X2 Pro** getestet. Beide verwenden die gleichen Belegungen, daher gelten – mit wenigen Ausnahmen – die folgenden Steuerungen.

### Allgemein
- **Start**: Pause
- **Select (OK)**: X
- **Zurück**: Kreis

### Zu Fuß
- **Charakterbewegung**: Steuerkreuz oder linker Joystick
- **Rennen**: Kreis
- **Springen**: X
- **Doppelsprung**: X + X
- **Angriff**: Quadrat
- **Interagieren**: Dreieck
- **Kamera bewegen**: Rechter Joystick
- **Zoom hinein**: Rechten Joystick nach vorne oder L2 drücken (stärker)
- **Zoom hinaus**: Rechten Joystick nach hinten ziehen
- **Nach oben schauen**: R2

### Im Fahrzeug
- **Beschleunigen**: R2 (progressiv) oder X
- **Bremsen**: L2 (progressiv) oder Kreis
- **Kamera wechseln**: L1
- **Hupe**: R1
- **Kamera bewegen**: Rechter Joystick
- **Nach hinten schauen**: Rechten Joystick nach hinten ziehen
- **Position zurücksetzen**: Select
